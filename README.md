O objetivo de longo prazo é atuar como **Engenheiro de Automação**, com formação em nível de mestrado, trabalhando na interseção entre **Controle, Automação Industrial e Sistemas Embarcados**.

Este repositório funciona como um **diário técnico de aprendizagem**, documentando meu progresso em programação, engenharia e pesquisa — não uma vitrine de projetos finalizados, mas um registro honesto de construção de conhecimento.
